## Prework - Text Color Changer

### App Description
The application changes the text of the 'Hello' label from red to orange after clicking the button and also outputs "Hello" into the console.

- [x] 1. User sees custom text in a label - Hello from {name}!
- [x] 2. User see's custom background color.
- [x] 3. User can tap a button to change the text color of the label.
